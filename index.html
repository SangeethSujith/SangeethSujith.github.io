Team VOS

Team VOS is an app created for the Vaidyaratnam Oushadhasala to ease their CRM work with various employees from managers, sales-person and medical representatives to view, report, enquire and manage employees using individual logins. Users and view customers, their purchase history and ledgers. Managers can assign tasks and send messages to their respective sales-persons. The whole project is made for android using React Native Framework. Code base is entirely in Javascript and is styled in CSS stylesheet.

Ayurware

Ayurware is a React Native app that helps the hospitals in consultation management. Doctors can also use the app to schedule appointments, accept appointments, and attend video calls with patients. It is a great way for patients to get the healthcare they need. Patients can simply use the app to find a doctor, book an appointment, and video conference with the doctor from the comfort of their own home.

The app is written in TypeScript and styled with Tailwind CSS.
